# خطة كورس أساسيات البرمجة بـ Python

## الجمهور المناسب
طلاب من 12 إلى 18 سنة أو مبتدئين تمامًا يريدون فهم البرمجة عمليًا.

## مدة الكورس
10 أسابيع. حصة لايف أسبوعية + تسجيل + واجب + كويز + مشروع تدريجي.

## مخرجات الطالب
- يكتب برامج Python بسيطة بدون نسخ ولصق.
- يفهم المتغيرات والشروط واللوبات والقوائم والدوال.
- يتعامل مع ملفات JSON و APIs ومقدمة SQL.
- يستخدم Git/GitHub لتسليم المشروع.
- يخرج بمشروع نهائي وشهادة وملف إنجاز.

## الخطة الأسبوعية
| الأسبوع | الموضوع | التطبيق |
|---|---|---|
| 1 | إعداد البيئة وأول برنامج | برنامج تعارف بسيط |
| 2 | المتغيرات والأنواع | آلة حاسبة |
| 3 | الشروط واللوبات | لعبة تخمين رقم |
| 4 | القوائم والقواميس | مدير مهام |
| 5 | الدوال والموديولات | تنظيم مشروع سابق |
| 6 | Debugging و Clean Code | إصلاح كود فيه أخطاء |
| 7 | OOP | نموذج طالب/كورس |
| 8 | Files و JSON و APIs | قراءة بيانات وحفظها |
| 9 | SQL Basics | جدول طلاب واستعلامات |
| 10 | Git/GitHub | المشروع النهائي والتسليم |

## شكل كل درس
1. تمهيد 5 دقائق.
2. شرح 20 دقيقة.
3. تطبيق مباشر 20 دقيقة.
4. تحدي قصير 10 دقائق.
5. واجب + كويز.
